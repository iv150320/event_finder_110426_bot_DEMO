# AGENTS.md — инструкции для opencode

## Проект
Event Finder Bot — Telegram-бот и фоновый сканер мероприятий (v9.3.0).
Директория: `/root/event_finder_110426_bot`

## Контекст между сессиями
Перед началом работы **всегда** читай `CONTEXT.md` — там история сессий, текущее состояние, известные проблемы и принятые решения.
После каждой значимой работы **обновляй** `CONTEXT.md`: добавляй запись в «История сессий» и обновляй «Последняя активность».

## Правила
- Язык комментариев и документации: русский
- Язык кода: английский (переменные, функции, классы)
- Не коммить изменения без явного запроса пользователя
- Перед запуском тестов убедись, что `.env` не содержит реальных секретов
- Используй `pytest -q` для тестов
- Синхронные HTTP-вызовы оборачивай в `asyncio.to_thread()`
- HTTP-сессии создавай через `_make_session()`, не используй глобальную shared-сессию

## Ключевые файлы
- `CONTEXT.md` — память между сессиями (читать/обновлять)
- `bot.py` — Telegram bot
- `scheduler.py` — фоновый сканер
- `event_search.py` — поиск и парсинг событий
- `database.py` — SQLite слой
- `notion_service.py` — Notion API
- `calendar_service.py` — ICS генерация
- `ics_server.py` — HTTP ICS server
